## 🚀 What I Build

- **Full-stack web apps** with Next.js (SSR, SSG, API routes) + Fastify backends deployed to production
- **Auth systems** with OTP flows, JWT, session handling, bcrypt, rate-limiting, and DDoS protection
- **Realtime & caching** using Redis (Upstash) for OTP storage and performance
- **Database-backed APIs** connecting Supabase/PostgreSQL with clean TypeScript schemas
- **Animated UIs** using GSAP for scroll-triggered and interactive experiences
