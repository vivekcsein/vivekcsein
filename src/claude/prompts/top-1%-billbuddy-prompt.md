# Prompt used in making BillBuddy

```
You are an elite, conversion-focused UI/UX Designer and a Senior Frontend Engineer. Your task is to build a visually stunning, high-converting, single-page landing page based on the context provided below.

### 1. CONTEXT MATRIX

- **Product/Service:** [e.g., BillBuddy - an AI-driven invoicing tool for freelancers]
- **Target Audience:** [e.g., Solo freelancers and agency founders who hate spending weekends chasing payments]
- **Primary Conversion Goal:** [e.g., Sign up for a 14-day free trial, no credit card required]
- **Aesthetic & Mood:** [e.g., Clean Minimalist / Premium B2B / Neo-Brutalist / Sleek Dark Mode]
- **Core Colors:** [Optional: Provide 2-3 hex codes or describe the palette, e.g., Deep slate background, obsidian cards, and a vibrant electric emerald green for CTAs]

### 2. ARCHITECTURAL REQUIREMENTS

Build the landing page in a single, fully responsive file using HTML, Tailwind CSS, and Lucide Icons (or vanilla SVGs). The layout must follow this conversion-focused structure:

1.  **Sticky Navigation:** Clean logo placeholder, 3-4 feature links, and a sharp, high-contrast primary CTA button.
2.  **Hero Section:**
    - An attention-grabbing, benefit-driven H1 headline (No AI fluff like "Revolutionize your workflow"—focus on the tangible outcome).
    - A persuasive subheadline addressing the core pain point.
    - A prominent dual-CTA block (Primary: high contrast; Secondary: ghost button).
    - An interactive, beautifully styled visual component mock-up below or to the side (styled completely via Tailwind to represent the product UI).
3.  **Social Proof Strip:** A modern, subtle marquee or row of "Trusted by" brand placeholders or a high-impact metric counters.
4.  **The Pain & Solution (PAS Framework):** A 3-column grid highlighting the user's current problems, immediately contrasted by how our product effortlessly solves them.
5.  **Feature Deep-Dive:** A alternating zebra layout (Image/UI on left, text on right, then flipped) showcasing the top 3 core features. Use polished card layouts with hover interactions.
6.  **Social Proof / Testimonials:** A modern masonry or horizontal grid of premium user testimonials with customer avatars, names, titles, and verified badges.
7.  **Pricing Section:** A clean 3-tier layout (Free/Starter, Pro, Enterprise) featuring a toggle for Monthly/Annual pricing (build the interactive toggle behavior using simple vanilla JavaScript). Highlight the "Most Popular" card with distinct borders and shadows.
8.  **Interactive FAQ Accordion:** 4-5 micro-targeted questions addressing common buyer friction points. Implement vanilla JS click behaviors to expand/collapse smoothly.
9.  **Final Urgency Call-to-Action (CTA):** A high-energy, centered banner wrapping up the final value proposition, a single clear input field or button, and low-friction guarantees (e.g., "No credit card required. 2-minute setup.").
10. **Footer:** Standard cleanly organized links (Product, Company, Legal), copyright notice, and social icons.

### 3. ANTI-AI VISUAL CONSTRAINTS (CRITICAL)

- **NO Clichés:** Absolutely no abstract purple-to-pink gradient meshes, overused generic tech circles, or meaningless floating blobs.
- **UI Components Over Stock Photography:** Where image placeholders are required, create rich, beautiful, interactive UI mockups using Tailwind (e.g., a modern dashboard card, a gorgeous graph, or a clean sleek input interface).
- **Micro-interactions:** Add smooth CSS transitions (`transition-all duration-300`) to every button hover, card elevation change, and navigational link.
- **Typography:** Maximize visual hierarchy using varied text sizing (`text-sm` up to `text-5xl` or `text-6xl` for hero), tracking (`tracking-tight`), and weights (`font-bold`, `font-medium`). Ensure contrast ratios are perfectly compliant for high readability.

Deliver the entire layout as a single, production-ready snippet of code. Take a deep breath, map out the UX hierarchy, and execute down to the pixel.
```
